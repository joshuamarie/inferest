# inferest

Title: ***Higer Level Interface for Statistical Inference***

This package attempts to re-imagine of what “grammar of statistical
inference” should look like, by performing statistical inference in
general, including:

1.  Simulation
2.  Bayesian inference
3.  Hypothesis testing

with `{tidyverse}` (`{ggplot2}` to be precise) paradigm.
